contrib/rsstool_php    PHP code that demonstrates how the feeds collected by RSStool can be used in own sites

contrib/rss2joomla.sh  script for adding content from RSS to Joomla! (replacement for --joomla option)
contrib/rss2phpbb3.sh  script for adding content from RSS to phpBB3 forum (replacement for --dragonfly option)

contrib/yt2phpfox.sh   script for adding videos from youtube to phpfox/video

contrib/xmlget         small tool for using XPath queries on XML documents
